# Restart Handoff (Docker install unblock)

Date: 2026-03-03

## Current blocker
`brew install --cask docker-desktop` failed due to stale symlink conflict:
- `/usr/local/bin/kubectl.docker` exists
- It is a broken symlink to removed `/Applications/Docker.app/...`

## Next steps after restart
Run from terminal:

```sh
rm /usr/local/bin/kubectl.docker
brew install --cask docker-desktop
open -a Docker
```

Wait until Docker says it is running, then verify:

```sh
docker version
```

## Then release services
From repo root:

```sh
./scripts/release-graphql.sh dev
./scripts/release-head.sh dev
```

## If brew errors again
If you see `already a Binary at '/usr/local/bin/<name>'`:
1. `ls -l /usr/local/bin/<name>`
2. remove stale/broken symlink
3. rerun install
